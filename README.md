[📕文档](https://docs.hualib.com/template/)

## 注意

此模板需要`EasyBot v1.3.3`以上的版本,不可在旧版本 EasyBot 上运行！

## 使用

```shell
git clone https://github.com/zkhssb/easybot-react-vite-template.git
cd easybot-react-vite-template
npm i
npm run build
```
