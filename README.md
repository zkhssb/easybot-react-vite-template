## 注意

此模板需要`EasyBot v1.3.4`以上的版本,不可在旧版本EasyBot上运行！

## 使用

```bash
git clone https://github.com/zkhssb/easybot-react-vite-template.git
cd easybot-react-vite-template
npm i
npm run build
```
